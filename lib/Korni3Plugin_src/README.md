# Korni3 Plugin

korni3plugin is a pulgin bridge for korni3 storage system

##Dependencies

- Qt 5.0 or higher.

## Build & Install

- `$mkdir build; cd build`
- `qmake ..`
- `make`
- `sudo make install`
